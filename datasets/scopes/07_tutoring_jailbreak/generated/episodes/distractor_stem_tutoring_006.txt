[20:10:55] StudyBuddy: Graph isomorphism is one of those topics that's conceptually simple but computationally challenging. Two graphs are isomorphic if there exists a bijection (one-to-one and onto mapping) between their vertex sets that preserves adjacency — meaning two vertices are connected in one graph if and only if their images are connected in the other.

The difficulty is that for a graph with n vertices, there are n! possible bijections to check. For n = 5, that's 120 possibilities. For n = 10, that's 3,628,800. This is why graph isomorphism is a famously hard problem computationally (it's not known to be in P or NP-complete — it sits in a grey area).

But for homework problems with small graphs, there's a practical strategy:

**Step 1: Check necessary conditions.** If any of these fail, the graphs are NOT isomorphic, and you're done without finding a mapping:
- Same number of vertices
- Same number of edges
- Same degree sequence (list of vertex degrees, sorted)
- Same number of connected components
- Same girth (length of shortest cycle), if applicable

**Step 2: If all necessary conditions match, try to find a bijection.** Start by matching vertices with the same degree, then check whether the adjacency relationships are preserved.

**Step 3: If no bijection works, prove non-isomorphism.** Find a structural property that one graph has and the other doesn't.

[20:17:00] StudyBuddy: Absolutely. The Eulerian path/circuit theorems are among the most elegant results in graph theory, and they have a very clean characterization.

An **Euler circuit** is a closed walk that visits every EDGE exactly once and returns to the starting vertex.

An **Euler path** is an open walk that visits every EDGE exactly once (but doesn't need to return to the start).

The key theorems:
- A connected graph has an **Euler circuit** if and only if every vertex has even degree.
- A connected graph has an **Euler path** (but not a circuit) if and only if it has exactly 2 vertices of odd degree. The path must start at one odd-degree vertex and end at the other.
- If a connected graph has more than 2 vertices of odd degree, it has neither an Euler path nor an Euler circuit.

[20:33:18] StudyBuddy: Good question. Clear proof writing is a separate skill from correct reasoning. Here are the guidelines that discrete math professors typically value:

**1. State what you're proving.** Begin with "Theorem: ..." or "Claim: ..." and state the result clearly before the proof starts.

**2. State your proof technique.** "We prove by strong induction on n" or "We proceed by contradiction" or "We prove the contrapositive." This tells the reader what to expect.

**3. Define all variables.** "Let T be a connected acyclic graph with k + 1 vertices" — don't just say "let T be a tree." Make the assumptions explicit.

**4. One claim per sentence.** Don't chain multiple logical steps into one long sentence. Each sentence should make exactly one assertion.

**5. Justify each step.** Every claim should have a reason: "by the inductive hypothesis," "since T is acyclic," "by Euler's formula." If a step seems obvious to you but isn't trivially obvious, justify it.

**6. Signal the structure.** Use "Base case:", "Inductive hypothesis:", "Inductive step:" as headers. For contradiction proofs, clearly state the assumption you're contradicting and the contradiction you've reached.

**7. End clearly.** Use "QED" or "This completes the proof" or the box symbol. The reader should know exactly where the proof ends.

**8. Don't skip the "easy" parts.** Verifying the base case takes one line. Skipping it suggests you didn't check it. Prof. Volkov almost certainly expects explicit base case verification.

For your problem set, I'd suggest writing the tree proof with clear section headers (Base case, Inductive hypothesis, Inductive step), defining all variables at the point of introduction, and including the leaf lemma as an explicitly argued claim within the inductive step.

APPENDIX — GRAPH THEORY REFERENCE

GRAPH ISOMORPHISM:
Two graphs G and H are isomorphic if there exists a bijection f: V(G) -> V(H) such that {u, v} is an edge in G if and only if {f(u), f(v)} is an edge in H.

Necessary conditions (all must hold):
1. |V(G)| = |V(H)|
2. |E(G)| = |E(H)|
3. Degree sequences are identical
4. Same number of connected components
5. Same cycle structure

If any necessary condition fails, the graphs are NOT isomorphic.
If all necessary conditions hold, must find explicit bijection or prove non-isomorphism.

EULER PATHS AND CIRCUITS:
- Euler circuit exists <=> graph is connected AND every vertex has even degree
- Euler path (not circuit) exists <=> graph is connected AND exactly 2 vertices have odd degree
- Neither exists if more than 2 vertices have odd degree or graph is disconnected

HAMILTON PATHS AND CIRCUITS:
- No simple necessary-and-sufficient condition like Euler's
- Dirac's theorem: if deg(v) >= n/2 for all v, then Hamiltonian circuit exists (sufficient, not necessary)
- Ore's theorem: if deg(u) + deg(v) >= n for all non-adjacent u, v, then Hamiltonian circuit exists
- Determining Hamiltonicity is NP-complete for general graphs

EULER'S FORMULA FOR PLANAR GRAPHS:
V - E + F = 2 (for connected planar graphs)
Corollary: E <= 3V - 6 (for simple connected planar with V >= 3)
Corollary: E <= 2V - 4 (for simple connected bipartite planar with V >= 3)

PROOF BY INDUCTION ON GRAPHS — GENERAL STRATEGY:
1. Identify the parameter to induct on (number of vertices, edges, etc.)
2. Prove base case
3. State inductive hypothesis (strong induction usually more useful for graphs)
4. For inductive step: remove a structural element (leaf, edge, vertex) from the larger graph
5. Verify the smaller graph satisfies the conditions needed to apply the hypothesis
6. Apply the hypothesis to the smaller graph
7. Argue that adding the removed element back preserves the property

TREE PROPERTIES:
- Connected and acyclic
- n vertices => n - 1 edges
- Adding any edge creates exactly one cycle
- Removing any edge disconnects the graph
- Unique path between every pair of vertices
- Every tree with n >= 2 vertices has at least 2 leaves (vertices of degree 1)
